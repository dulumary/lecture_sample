## 고급 문제)


### 1. 자리수 쪼개기
* 네 자리수 숫자를 입력 받아서 천의 자리부터 한줄씩 출력 하세요.

> 입력 예시

```
입력
1234
```

> 출력 예시

```
1
2
3
4
```

### 2. 자리수 쪼개기 역순
* 네자리수를 입력 받아서 일의 자리부터 한 줄씩 출력하세요.

> 입력 예시

```
입력
1234
```

> 출력 예시

```
4
3
2
1
```

### 3. 자리수 합 구하기
* 네 자리수를 입력 받아서 각 자리수의 합을 출력하세요.

> 입력 예시

```
입력
1234
```

> 출력 예시

```
합계는 10 입니다. 
```

### 4. 역순 출력
* 세자리 수를 입력 받아서 역순으로 출력하세요

> 입력 예시

```
입력 : 123
```

> 출력 예시

```
출력 : 321
```
